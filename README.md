# GWA Demo Test Repository

This repository is used to test the GitHub Workflow Agents (GWA) system end-to-end.

## Features to Test
- GitHub Projects integration
- Column-based workflow triggers
- Multi-agent swarm architecture
- Planning templates
- Question/Answer flow

## Test Workflow
1. Create issue → Added to Project Todo
2. Move to Planning → Claude starts planning session
3. Move to In Progress → Claude implements solution
4. Move to QA → Playwright tests run
5. Move to Review → Awaiting human review
6. Move to Done → Cleanup and deployment
